## Releasing

ownCloud Web no longer has a standalone release process. Its source lives in-tree under
`web/` in the [ocis repository](https://github.com/owncloud/ocis) and is built and released
as part of oCIS itself. See [oCIS Web service / Releasing](/services/web/releasing) for the
current process.

The final standalone release of ownCloud Web was `v12.5.0` (also tagged `last-agpl-release`),
after which the [owncloud/web](https://github.com/owncloud/web) repository was archived.
